# 📄 Currículo Profissional - Lucas Ferreira

Bem-vindo ao repositório do meu currículo profissional online! Este projeto é uma página web responsiva e minimalista, desenvolvida com foco em performance e exportação perfeita para PDF, apresentando minha trajetória como **Desenvolvedor Front-End**.

🔗 **[Acesse o projeto ao vivo aqui](https://curriculolucas.ferreiradev.com/)** 

## 🎯 Sobre o Projeto

Criei este currículo através de código para ter total controle sobre o design, a escaneabilidade e a hierarquia da informação. Sou um desenvolvedor que se importa em criar interfaces web modernas, responsivas e performáticas, e este projeto reflete meu cuidado com a estrutura e estilização limpa.

## 🛠️ Tecnologias Utilizadas (Neste Repositório)

- **HTML5** (Estrutura semântica e acessível)
- **CSS3** (Flexbox, Grid, Media Queries e customização avançada com `@media print`)

## ✨ Funcionalidades e Diferenciais  

- 📱 **Design Responsivo:** Layout adaptável para Desktops, Tablets e Smartphones.
- 🖨️ **Otimizado para Impressão (PDF):** Estilização específica configurada para gerar um PDF limpo, cravado em 1 página A4, ocultando elementos desnecessários na hora da exportação.
- ⚡ **Zero Dependências:** Código puro, leve e de carregamento instantâneo.
- 🎯 **Foco em UI/UX:** Estrutura pensada para facilitar a leitura rápida por recrutadores e líderes técnicos.
